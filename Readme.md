# LNMP 自动安装脚本

> V 1.1.0

最近总是搞LNMP环境测试，每次都手动搞环境，觉得太麻烦了，抽了一天时间，写了这个。

现在功能还是比较弱的，只支持 CentOS，实际测试的只有 CentOS 7.4，后面有时间再完善。

## 软件包的版本

采用的都是当前最新稳定版。

- MySQL 5.7.26

- PHP 7.1.30

- Tengine 2.3.1
